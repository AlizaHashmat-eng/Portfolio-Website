Simple ChatBot Application

Project Description

The Simple ChatBot Application is a lightweight and flexible platform designed to simulate basic human-to-machine interactions. Developed with a combination of modern front-end and back-end technologies, this project serves as an example of how to integrate chatbot functionality into web applications. The source code is easy to understand and customize, making it an excellent tool for learning and extending features.

Technologies Used

Front-End

  • HTML5
  • CSS
  • JavaScript
  • jQuery
  • Ajax
  • Bootstrap
  • AdminLTE Template

Back-End

  • PHP (version 8.0.2)
  • MySQL Database
  • MySQLi OOP

Key Features

  • Public Features:
    • Chatbot Conversations: Engage with a simple and interactive chatbot interface.
  • Admin Panel:
    • Home: View system summary and statistics.
    • Manage System Settings: Configure chatbot behaviors and appearance.
    • Manage Responses: Add, edit, or remove chatbot responses for specific queries.
    • Manage Unanswered Queries: Review and address queries the chatbot could not resolve.

Learning Outcomes

This project demonstrates how to create a Simple ChatBot that can be seamlessly added as a feature to other projects. The implementation showcases modern development practices, and its source code is designed to be easy to understand, adapt, and expand for diverse applications.
